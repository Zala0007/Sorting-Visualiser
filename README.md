# 🔢 Sorting Visualizer

An interactive and animated visualizer for popular sorting algorithms, built using **React** and **JavaScript**. This project demonstrates how various sorting algorithms work step-by-step with intuitive UI and animations.

---

## 🚀 Features

- 📊 Visual representation of sorting process using colored bars
- ⏸ Pause / Resume functionality
- 🔁 Reset the sorting at any time
- ⏱ Adjustable sorting speed
- 🔢 Choose number of elements in the array
- 🔄 Algorithms included:
  - Bubble Sort
  - Selection Sort
  - Insertion Sort
  - Merge Sort
  - Quick Sort (optional)
- 📈 Display of statistics: comparisons, swaps, time taken

---

## 🛠 Tech Stack

- **Frontend**: React, JavaScript/TypeScript
- **Styling**: CSS / Tailwind CSS (if used)
- **State Management**: React hooks (`useState`, `useEffect`)
- **Deployment**: GitHub Pages / Vercel / Netlify
